## Q1: 타겟 사용자 및 언어

앱의 주요 타겟 사용자와 UI 언어는 무엇인가요?

A) 한국어 전용 (한국 사용자 대상)
B) 영어 전용 (글로벌 사용자 대상)
C) 한국어 + 영어 다국어 지원
D) 영어 기본, 한국어 추후 지원

[Answer]: B (영어 전용, 추후 다국어 지원 예정)

## Q2: 인증 방식

Firebase Authentication에서 어떤 로그인 방식을 지원할까요?

A) Google 로그인만
B) Google + 이메일/비밀번호
C) Google + 이메일 + Apple 로그인
D) 익명 로그인 (게스트 모드) + Google 로그인

[Answer]: D (익명 + Google 로그인)

## Q3: 데모 시나리오 우선순위

해커톤 데모(4분)에서 가장 강조할 기능은 무엇인가요? (복수 선택 가능)

A) 음성 기상 알람 + 대화형 인터랙션
B) 실시간 비디오 인증 (행동 인식)
C) 일일 리포트 + AI 분석
D) 전체 루틴 플로우 시연

[Answer]: A, B (음성 기상 알람 + 실시간 비디오 인증)

## Q4: MVP 범위 (1주 목표)

1주 안에 완성할 MVP 범위를 정해주세요:

A) 핵심만: 음성 알람 + 루틴 진행 + 비디오 인증 (Req 3, 4, 5)
B) 핵심 + 리포트: 위 + 일일 리포트 (Req 3, 4, 5, 6)
C) 핵심 + 설정: 위 + 루틴 설정 UI (Req 2, 3, 4, 5, 6)
D) 전체 기능 (모든 요구사항)

[Answer]: C (핵심 + 설정 UI: Req 2, 3, 4, 5, 6)

## Q5: 비디오 인증 행동 종류

실시간 비디오 인증에서 인식할 행동 종류를 정해주세요:

A) 간단한 행동만 (물 마시기, 손 흔들기)
B) 운동 행동 포함 (스트레칭, 버피 자세)
C) 장소 인식 포함 (현관문 앞, 운동 공간)
D) 사용자가 자유롭게 정의 (텍스트 설명 기반)

[Answer]: D (사용자 자유 정의, 안 되면 수정 예정)

## Q6: 데이터 보존 기간

사용자 루틴 기록 데이터를 얼마나 보존할까요?

A) 7일 (최근 1주)
B) 30일 (최근 1개월)
C) 90일 (최근 3개월)
D) 무제한 (사용자가 삭제할 때까지)

[Answer]: D (무제한, 최소 1년 - "1년 전 당신과 비교" 기능 위해)

## Q7: 오프라인 지원 범위

PWA 오프라인 모드에서 지원할 기능은?

A) 기본 UI만 (데이터 조회 불가)
B) 루틴 목록 조회 + 수동 체크

## Q8: IaC 도구 선택

Infrastructure as Code 도구로 무엇을 사용할까요?
