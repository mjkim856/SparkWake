## 📝 Recording Checklist

### Before Recording
- [ ] Routines set up (Stretching, Yoga with YouTube, Hand Wave mission)
- [ ] Good lighting for camera verification
- [ ] Quiet environment for clear audio
- [ ] Mobile in portrait mode
- [ ] Screen recorder ready

### During Recording
- [ ] Speak clearly and at moderate pace
- [ ] Wait for AI responses (don't rush)
- [ ] Make barge-in obvious (interrupt mid-sentence!)
- [ ] Show enthusiasm (it's a morning app!)

### After Recording
- [ ] Add subtitles for all speech
- [ ] Add feature highlight subtitles (✅ Barge-in, etc.)
- [ ] Add logo/branding overlays
- [ ] Check total time ≤ 4 minutes

## 💡 Pro Tips

1. **Barge-in must be OBVIOUS** - AI should be mid-sentence when you interrupt
2. **Show the AI indicator** - Waveform or speaking icon helps visualize
3. **Subtitles are essential** - Judges might watch on mute
4. **Energy matters** - It's a morning motivation app, be upbeat!
5. **Keep it under 4 min** - Respect the time limit
